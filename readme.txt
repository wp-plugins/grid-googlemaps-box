GoogleMaps Box for [Grid](http://wordpress.org/plugins/grid/ "Grid Landingpageeditor").

== Description ==

Provides a Grid Box that uses GoogleMaps V3. Needs Grid v1.3 or higher.

== Installation ==

1. Upload `grid-googlemaps-box.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `grid-gmaps-box` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Put GoogleMaps Boxes on your Grid Landingpages
